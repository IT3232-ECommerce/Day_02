# Day_02



## API Endpoints
| Method | Endpoint     | Description |
|--------|-------------|-------------|
| GET    | `/app/msg`  | Returns a greeting message |
| GET    | `/app/name` | Returns the application's name |

## Example Response
### `GET /app/msg`
```
Hello Spring Boot...
```

### `GET /app/name`
```
My name is Spring Boot
```
